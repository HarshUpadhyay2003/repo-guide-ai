# Security Policy

## Supported Versions

We currently support the following versions of RepoGuideAI with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

Security is a top priority for RepoGuideAI. We appreciate your efforts to responsibly disclose your findings.

If you discover a potential security vulnerability in this project, **please do not publish it in the public GitHub issue tracker.** Instead, please report it privately via GitHub Security Advisories:

1. Go to the "Security" tab of the RepoGuideAI GitHub repository.
2. Click on "Report a vulnerability" under "Private vulnerability reporting".
3. Provide a detailed summary of the vulnerability, including steps to reproduce. We will review the submission and respond promptly with an acknowledgment and an estimated timeline for the fix.
